Чистая Установка

Пошаговый и безопасный гайд по установке Windows. Проверяйте официальные ресурсы и делайте резервные копии.

1. Подготовка

  • Резервное копирование важных данных (внешний диск, облако).
  • Проверка совместимости: версия Windows, требования к TPM/UEFI, объём диска и RAM.
  • Скачивайте установочный образ только с официального сайта Microsoft.
  • Подготовьте пустую USB-флешку 8 GB+ (все данные на ней будут удалены).

2. Создание загрузочной флешки

  • Рекомендуемые инструменты: официальный Media Creation Tool (Windows) или balenaEtcher (кроссплатформенный).
  • Проверьте контрольную сумму ISO (если доступна) для предотвращения повреждённого/подменённого образа.
  • Инструкция: записать ISO на флешку, сделать её загрузочной (ruled by chosen tool).
Безопасность: не загружайте ISO с неизвестных сайтов — риск малвари и бэкдоров.

3. BIOS / UEFI

  • Войдите в BIOS/UEFI (обычно F2, Del, Esc — зависит от производителя).
  • Убедитесь, что режим загрузки — UEFI (рекомендуется) или Legacy по необходимости.
  • Отключите Secure Boot только если это явно требуется для старого оборудования; по возможности оставьте включённым.
  • Настройте загрузку с USB в приоритете.

4. Установка Windows

  1. Загрузитесь с флешки и следуйте мастеру установки.
  2. При выборе диска: при чистой установке можно удалить старые разделы (внимательно) и создать новые.
  3. Если у вас SSD — убедитесь, что в настройках включён AHCI и что диск использует GPT (для UEFI).
  4. После завершения — установите драйверы с сайта производителя и обновления через Windows Update.

5. Постинсталляционные шаги и безопасность

  • Включите BitLocker (если доступен) и создайте резервную копию ключа восстановления.
